# LUCID

**LUCID** is my capstone short film project, created as a narrative film and presented through polished web-based documentation.  
The project explores the unstable line between dreams and waking life through suspense, surreal visual details, and cinematic atmosphere.

## Overview

This repository serves as the main documentation hub for **LUCID**.  
It is designed to present the project as both a creative work and a technical production artifact, with a front-facing GitHub Pages site and additional process documentation.

This project includes:
- a polished project website
- an overview of the film’s concept and goals
- documentation of pre-production, production, and post-production
- visuals and media from the project
- evidence of technical and creative workflow

## Logline

## Creative Goals

The main goals of **LUCID** were to:

- create a visually immersive short film with a dreamlike atmosphere
- balance suspense with cinematic beauty
- build a consistent visual identity across multiple locations and scenes
- refine the project through editing, sound, and color
- present the final work in a professional, portfolio-ready format

## Highlights

### Visual Direction
The visual language of **LUCID** was built around contrast: calm, cinematic beauty against tension, uncertainty, and distorted reality.

### Production
The film required planning locations, organizing shoot days, directing talent, and making technical decisions on set to support the tone of the story.

### Post-Production
Post-production focused on shaping rhythm, suspense, sound, and visual consistency through editing and color refinement.

## What I Learned

**LUCID** pushed me to think more deeply about the full filmmaking process, from idea development and planning to execution and final presentation.  
It strengthened my ability to balance creative vision with technical workflow, and to present a film project professionally for audiences beyond the classroom.
